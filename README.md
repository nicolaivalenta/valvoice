# ValVoice

Local-first dictation for macOS. Press caps lock, speak, release — transcribed text appears at your cursor.

## Features

- Fully offline transcription via Parakeet (Apple Silicon)
- Instant word replacements / dictionary
- Yellow vibe

## Building

Requires Xcode 15+ on Apple Silicon macOS 13+.

```
open VoiceInk.xcodeproj
```

Set your development team in Signing & Capabilities, then Build.

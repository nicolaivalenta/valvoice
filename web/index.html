How to install

Three steps and you're dictating.

  1. Copy the install command
    curl -fsSL https://valvoice.app/install | bash
  2. Open Terminal
    Press ⌘Space, type Terminal, then press ↵.
  3. Paste and run
    Press ⌘V to paste, then ↵ to run. ValVoice installs itself and opens automatically.
  • 100% offline
